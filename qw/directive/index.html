# 全局指令

# v-fullHeight

  • 说明

    应用对象：Element（普通元素，不可用于Vue组件）

    作用：将应用该指令的元素高度设为视口高度减去元素距离顶部的高度

  • 使用方法

    <div v-fullHeight></div>
