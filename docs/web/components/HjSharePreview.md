### Attributes

参数|说明|类型|可选值|默认值
:-|:-|:-|:-|:-
isShowApplets | 是否显示小程序预览效果 | Boolean | - | true
currentTab | 当前预览Tab | String | friends/friendsCircle/Applets | friends
