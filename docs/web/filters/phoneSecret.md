# phoneSecret

在加密模式下对电话进行加密

### 加密规则

- 号码小于等于5位时不隐藏

- 号码大于等于6位小于等于11位时隐藏中间四位

- 号码大于11位时前面保留三位，后面保留四位，中间全部隐藏

### 参数

- {String} value 要加密的电话（经过后端加过密的或者是正常的电话号码）

- {Boolean} arg 是否要进行加密 不指定值的情况下使用系统设置的

### 返回值  

- {String} res 处理过后的电话

### 示例

```js
{{ item | phoneSecret }}

eg: 

{{ 17354554688 | phoneSecret }} 处理后返回  173****4688

{{ 1735 | phoneSecret }} 处理后返回  1735
```
